# Virtual User interview

## Obawy uzytkownika

- Agent nie moze publikowac tresci niskiej jakosci albo obietnic pewnej przyszlosci.
- Reklamy nie moga przepalic budzetu ani naruszyc polityk platform.
- Strona nie moze sama zmieniac legal/auth/checkout bez kontroli.

## Oczekiwanie

Autopilot ma dzialac sam, ale operator musi widziec status, blokady, koszty i powody decyzji.

## Polish summary

Pelna autonomia jest akceptowalna tylko wtedy, gdy jest widoczna, limitowana i audytowalna.
